# GRAPHQL_NODEJS_SAMPLE

## Setup

Install:

```sh
$ npm ci
```

Run the server:

```sh
$ npm run start
```

Open GUI

```sh
http://localhost:4200/playground
```

Run test:

```sh
$ npm run test
```
